# Sistema de Estoque e Vendas

Projeto desenvolvido para o P.I.

## Equipe
Helena P. Bequi 

## Objetivo
Criar um sistema para gerenciamento de estoque e vendas de lojas de roupas.

## Funcionalidades
- Cadastro de produtos
- Controle de estoque
- Registro de vendas
- Relatórios
- Alertas de estoque baixo

## Tecnologias
- HTML
- JavaScript
- Node.js
- MySQL
